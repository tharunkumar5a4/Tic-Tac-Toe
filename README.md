Tic Tac Toe — Team Project

Play Tic Tac Toe online with two players or against a simple AI.
Tracks scores for X, O, and draws.

⚡ Features

Two-player and AI mode

Scoreboard for wins/draws

Restart button & R key shortcut

Responsive design for mobile & desktop

Highlights winning combination

🛠 Tech

HTML5 – Game structure

CSS3 – Styling and responsive layout

JavaScript – Game logic and AI

▶️ How to Play

Click cells to place X or O.

Toggle Play vs Computer to play against AI.

Board shows current turn, wins, and draws.

Click Restart or press R to reset.
